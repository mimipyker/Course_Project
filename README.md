# Курсовая работа по ООП 
Простое десктопное приложение для учета личных финансов на C# + WPF (.NET).
Позволяет добавлять и удалять транзакции, искать записи по названию и категории, считать доходы/расходы и текущий баланс, а также сохранять данные в data.json.

Возможности
1. добавление доходов и расходов;
2. удаление выбранной транзакции;
3. поиск по транзакциям в реальном времени;
4. отображение общей суммы доходов, расходов и итогового баланса;
5. сохранение и загрузка данных из JSON.
Технологии
1. C#
2. WPF
3. .NET
4. System.Text.Json
